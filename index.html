The DormBell eliminates the need for knocking and provides clear, customizable notifications for residents.</p>

            <h2>Step-by-Step Instructions</h2>


            <div class="step">

                <h3>Step 1: Assemble the Components</h3>

                <ul>

                    <li>Place the Arduino Uno on the breadboard.</li>

                    <li>Connect the ultrasonic sensor:

                        <ul>

                            <li>VCC to 5V</li>

                            <li>GND to GND</li>

                            <li>TRIG to Pin 9</li>

                            <li>ECHO to Pin 8</li>

                        </ul>

                    </li>

                    <li>Connect the LCD display using the I2C module:

                        <ul>

                            <li>GND to GND</li>

                            <li>VCC to 5V</li>

                            <li>SDA to A4</li>

                            <li>SCL to A5</li>

                        </ul>

                    </li>

                    <li>Attach the buzzer:

                        <ul>

                            <li>Positive terminal to Pin 7</li>

                            <li>Negative terminal to GND</li>

                        </ul>

                    </li>

                    <li>Wire the dip switch to digital pins (e.g., 3, 4, 5, 6).</li>

                </ul>

            </div>


            <div class="step">

                <h3>Step 2: Upload the Code</h3>

                <p>Copy the following code and upload it to your Arduino Uno using the Arduino IDE:</p>

                <div class="code">

                    #include <Wire.h>\n#include <LiquidCrystal_I2C.h>\nconst int trigPin = 9;\nconst int echoPin = 8;\nconst int buzzer = 7;\nconst int dipPins[] = {3, 4, 5, 6};\nLiquidCrystal_I2C lcd(0x27, 16, 2);\nvoid setup() {\npinMode(trigPin, OUTPUT);\npinMode(echoPin, INPUT);\npinMode(buzzer, OUTPUT);\nfor (int i = 0; i < 4; i++) {\npinMode(dipPins[i], INPUT);}\nlcd.begin(16, 2);\nlcd.backlight();}\nvoid loop() {// Ultrasonic sensor logic\n}

                </div>

            </div>


            <div class="step">

                <h3>Step 3: Test and Debug</h3>

                <ul>

                    <li>Power the Arduino and move your hand in front of the ultrasonic sensor.</li>

                    <li>Check that the buzzer activates and the LCD displays the correct message.</li>

                    <li>Adjust the dip switch positions to verify message customization.</li>

                </ul>

            </div>
